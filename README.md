### 📄 Pages

1. `pages/StartPage.tsx`: 시작 페이지(로그인 전 메인)
2. `pages/LoginPage.tsx`: 로그지 페이지
3. `JoinPage.tsx`: 회원가입 패이지

5. `MemberPage.tsx`: 회원 페이지(로그인 후 매인)
